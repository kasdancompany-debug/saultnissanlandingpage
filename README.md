# Sault Nissan — QR Landing Page

A mobile-first landing page for QR code scans. Visitors get one-tap links to the Sault Nissan website and social profiles.

## Links included

| Platform  | URL |
|-----------|-----|
| Website   | https://www.saultnissan.ca |
| Facebook  | https://www.facebook.com/saultnissan |
| Instagram | https://www.instagram.com/saultnissan |
| TikTok    | https://www.tiktok.com/@saultnissan |

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deploy

This is a single static HTML file — no build step required. Host it on any static host:

- **Netlify** — drag the folder onto [app.netlify.com/drop](https://app.netlify.com/drop)
- **Vercel** — `npx vercel`
- **GitHub Pages** — push to a repo and enable Pages from the root

## QR code

Once deployed, generate a QR code pointing to your live URL (e.g. `https://links.saultnissan.ca`). Free tools: [qr-code-generator.com](https://www.qr-code-generator.com/) or Canva.
